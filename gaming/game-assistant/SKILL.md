泛用游戏待办记录 + 最强蜗牛供奉计时器

---

## 明日方舟待办

### 记录格式

```
### 明日方舟待办
- 干员名：任务描述 | 状态
```

状态标签：
- `pending` - 待完成
- `in_progress` - 进行中
- `completed` - 已完成

## 最强蜗牛供奉计时器

### 活动周期

- 活动顺序：抽奖周 → 许愿周 → 供奉周（每3周循环一次）
- 活动时间：每周五 08:00 → 下周五 04:00

### 贡品收获时间

| 贡品 | 收获时间 |
|------|----------|
| 龙珠 | 43小时7分9秒 |
| 大灵魂 | 5小时37分4秒 |

> ⚠️ 贡品收获时间因玩家加速光环不同而异，应以 data.json 中记录的用户实际时间为准

### ⚠️ 重要：多贡品排队计算

**9个贡品是依次排队收获的，不是同时收获！**

❌ 错误算法：只算一个贡品的时间
✅ 正确算法：所有贡品收获时间相加

### 计算公式

```
最晚放置时间 = 供奉周开始时间 - 所有贡品收获时间之和
```

### 供奉加速

实际供奉时间 = 灵魂供奉时间 / (1 + 供奉加速%)

| 加速来源 | 加速效果 |
|----------|----------|
| 绿贵神龙雷达满星 | +9%（龙珠 155229秒 → 143604秒） |
| 巨龙基因"龙之契约" | -3600秒 |
| 哭泣的男孩 | +1.2% |
| 死亡圣器VII | +1.2% |
| 佛牙舍利 | +4% |
| 玛雅头骨水晶 | +6.4% |
| 普罗米修斯火炬 | +8.8% |
| 司母戊鼎 | +8.8% |

## 附录：明日方舟抽卡计算

### 抽卡资源

| 资源 | 换算 |
|-----|------|
| 合成玉 | 600 = 1抽 |
| 源石 | 1 = 180合成玉 = 0.3抽 |
| 寻访凭证 | 1 = 1抽 |

### 计算公式

```
总抽数 = 合成玉/600 + 源石×0.3 + 寻访凭证
```
