# 个人任务管理检查清单

## 添加任务前
- [ ] openid 已在 user-mapping.json 中匹配到 assignee
- [ ] 任务描述简洁明确
- [ ] 开始时间已确认（默认今天）
- [ ] ID 无冲突（12位时间戳）

## 添加进展前
- [ ] 任务已找到（按 id 或内容模糊匹配）
- [ ] 进展内容包含具体日期
- [ ] 相对时间词已转为绝对日期

## 完成任务前
- [ ] 确认用户确实要标记为完成
- [ ] completed 时间自动填入

## 删除任务前
- [ ] 确认用户确实要删除
- [ ] 删除后不可恢复

## 查看任务时
- [ ] 默认只看当前用户的任务
- [ ] 已完成任务只显示最近7天
