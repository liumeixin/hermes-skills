# 封面模板选择决策树

## 步骤 1：确认平台

| 用户说 | 平台 | 尺寸 |
|--------|------|------|
| 微信/公众号/WX | 微信公众号 | 1550×660 或 1920×1080 |
| 小红书/XHS | 小红书 | 1080×1440 或 1080×1691 |
| 知乎 | 知乎 | 1550×660 |
| 都要 | 双平台 | 生成两个版本 |

## 步骤 2：确认风格来源

```
用户指定具体风格名？
├── 是 → 在下面查找
│   ├── 原创模板（f9-wx/xhs-*） → 直接使用 templates-original/
│   ├── Design Systems 风格 → 查 54 个风格列表
│   └── HTML Templates 风格 → 查 33 个风格列表
└── 否 → 使用默认模板（f9-wx/xhs）
```

## 步骤 3：生成封面

### 原创模板
直接修改 HTML 文件中的文字内容，然后截图。

### Design Systems 风格
1. 从 `design-systems/` 目录加载对应风格的颜色/字体规范
2. 生成新 HTML 文件
3. 截图

### HTML Templates 风格
1. 从 `html-templates/` 目录加载对应风格
2. 生成新 HTML 文件
3. 截图

## 常用风格快速参考

### 深色系（科技/AI 话题）
- linear-app：极简深色，紫罗兰强调
- cursor：深色科技
- studio：黑底+电黄
- ollama：终端风格单色

### 浅色系（生活/家居话题）
- notion：暖白极简
- vercel：黑白极简
- claude：暖色调 parchment 米色

### 彩色系（创意/设计话题）
- figma：多彩创意
- airbnb：温暖珊瑚色
- posthog：趣味绿品牌

### 特殊风格
- nowill-design：橙黑科技风（原创模板）
