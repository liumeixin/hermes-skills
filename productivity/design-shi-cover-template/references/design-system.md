# 封面设计规范

## 配色方案

### 深色科技风（默认）
```
背景: #121212
强调: #FF5A1F
文字: #FFFFFF
次要文字: #AAAAAA
装饰: #333333
```

### 极简白底
```
背景: #FFFFFF
强调: #FF5A1F
文字: #1A1A1A
次要文字: #666666
装饰: #E0E0E0
```

### 渐变科技
```
背景: linear-gradient(135deg, #121212, #1a1a2e)
强调: #FF5A1F
文字: #FFFFFF
```

## 字体规范

### 系统字体栈（可商用）
```css
--font-primary: system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
--font-mono: "SF Mono", "Fira Code", "Consolas", monospace;
```

### 字号层级
| 元素 | 最小字号 | 推荐字号 | 字重 |
|------|---------|---------|------|
| 主标题 | 36px | 48-72px | 700-900 |
| 副标题 | 18px | 24-32px | 400-500 |
| 正文 | 14px | 16-20px | 400 |
| 标签 | 12px | 14-16px | 500 |

### 行高
- 标题：1.1-1.2
- 正文：1.4-1.6

---

## 间距系统（4px基准）

| 层级 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 内联元素 |
| sm | 8px | 标签内边距 |
| md | 16px | 组件内部 |
| lg | 24px | 组件之间 |
| xl | 32px | 区块之间 |
| 2xl | 48px | 大区块之间 |

---

## 圆角规范

| 元素 | 圆角值 |
|------|--------|
| 按钮 | 4-8px |
| 卡片 | 8-12px |
| 标签 | 4px |
| 大卡片 | 16px |

---

## 阴影规范

### 推荐（轻阴影）
```css
box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1);
```

### 禁止
- 硬阴影（hard drop shadow）
- 过重阴影（>0.3 opacity）

## 封面结构模板

### 横版（16:9，1280×720）
```
┌────────────────────────────────────┐
│  [标签/分类]                       │
│                                    │
│  [主标题]                          │
│  [副标题]                          │
│                                    │
│  [装饰元素]              [署名]    │
└────────────────────────────────────┘
```

### 竖版（3:4，1080×1440）
```
┌──────────────┐
│  [标签]      │
│              │
│  [主标题]    │
│              │
│  [副标题]    │
│              │
│  [装饰]      │
│              │
│  [署名]      │
└──────────────┘
```

## 截图规范

### Playwright 配置
```python
# 横版
viewport = {'width': 1280, 'height': 720}

# 竖版（小红书）
viewport = {'width': 1080, 'height': 1440}

# 等待时间
wait_for_timeout(800)  # 确保字体加载
```

### 输出要求
- 格式：PNG
- 无滚动条
- 一屏显示完整内容
