# 快速风格选择指南

## 按场景选择

### 信息图/教程步骤图
→ **米色暖系（warm）**
- 背景：#f5f0e8
- 强调色：#8b6f47（棕褐）
- 适合：公众号配图、教程步骤图、微信群精华摘要

### 数据卡片/金句图
→ **深色极简（dark）**
- 背景：#0d0d0d
- 强调色：#4ade80（荧光绿）
- 适合：数据卡片、金句图、朋友圈传播

### 工具对比/方案对比
→ **双列对比（grid）**
- 背景：#1a1a2e
- 左右分栏，✓/✗ 标记
- 适合：工具对比、方案对比、优劣势对照

### 社交媒体封面
→ **毛玻璃 + Mesh Gradient（艺术风格）**
- 深色背景 + 多色径向光斑 + 毛玻璃卡片
- 适合：小红书/公众号封面、社交媒体传播图

## 配色方案速查

### 科技紫（NAS/Docker/效率工具）
- 光斑色：#ff6b6b, #4ecdc4, #a29bfe, #fd79a8
- 渐变高亮：#4ecdc4 → #a8e6cf

### 活力暖（生活/家电/数码）
- 光斑色：#ff6b6b, #feca57, #ff9ff3, #f368e0
- 渐变高亮：#feca57 → #ff9ff3

### 深海蓝（AI/软件/技术教程）
- 光斑色：#0984e3, #00cec9, #6c5ce7, #74b9ff
- 渐变高亮：#00cec9 → #74b9ff

### 森林绿（环保/健康/自然）
- 光斑色：#00b894, #55efc4, #a29bfe, #fd79a8
- 渐变高亮：#55efc4 → #a29bfe
