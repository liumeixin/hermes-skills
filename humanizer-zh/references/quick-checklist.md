# Humanizer-zh 快速检查清单

改写完成后逐项核对：

## 语言层面
- [ ] 连续三个句子长度相同？→ 打断其中一个
- [ ] 段落以简洁单行结尾？→ 变换结尾方式
- [ ] 揭示前有破折号？→ 删除它
- [ ] 解释了隐喻或比喻？→ 信任读者，删除解释
- [ ] 使用了"此外""然而""至关重要"等连接词？→ 考虑删除
- [ ] 三段式列举？→ 改为两项或四项

## 模式层面
- [ ] 还残留"标志着""不可磨灭""充满活力"？→ 删除或降调
- [ ] "不仅...而且..." 结构？→ 改为直接陈述
- [ ] "为了实现这一目标"→ "为了" / "做 X"
- [ ] "希望这对您有帮助"？→ 删除
- [ ] 段落结尾像金句/格言？→ 重写，去掉可引用感
- [ ] 过度使用破折号制造戏剧感？→ 改用逗号

## 节奏层面
- [ ] 是否混合了长短句？
- [ ] 段落长度是否有变化？
- [ ] 最后一句是否是总结性陈述？→ 改为留白或具体场景

## 评分标准
| 维度 | 及格 | 优秀 |
|------|------|------|
| 直接性 | 6/10 | 8/10+ |
| 节奏 | 6/10 | 8/10+ |
| 信任度 | 7/10 | 9/10+ |
| 真实性 | 6/10 | 8/10+ |
| 精炼度 | 7/10 | 9/10+ |
| **总分** | **32/50** | **42/50+** |
