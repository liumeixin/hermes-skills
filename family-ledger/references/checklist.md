# 家庭账本检查清单

## 记账前检查
- [ ] openid 已在 user-mapping.json 中匹配到显示名
- [ ] 日期格式为 YYYY-MM-DD
- [ ] 金额为数字（单位：元）
- [ ] 账本类型已自动判断（日常/人情/借款）

## 日常账特殊检查
- [ ] 备注中是否包含房屋名称 → 需确认后归类为"房屋维护"
- [ ] 教育类别 → 是否已关联孩子信息
- [ ] 收入/支出类别是否在允许列表中

## 人情账特殊检查
- [ ] giver 和 target 是否明确
- [ ] 压岁钱是否关联了 giver_child

## 借款账特殊检查
- [ ] direction 是否正确（借出/借入/报销给我/我报销）
- [ ] status 是否与操作匹配
- [ ] due_date 是否合理

## 查询前检查
- [ ] 时间范围是否明确（默认：当月）
- [ ] 是否需要模糊匹配
